# eetrade — Fresh Produce & Bales E-Commerce Landing Page

A modern and animated e-commerce landing page for **eetrade**, a fresh produce and bales trading platform offering 100% organic fruits, vegetables, and high-quality storage bales with 24-hour delivery.

## 🛠️ Tech Stack

- **HTML5 & CSS3** — semantic one-page structure and custom styling
- **Bootstrap 5.3** — responsive grid and components
- **Font Awesome 6.4** — icon set
- **Cairo + Poppins** — Google Fonts dual typography
- **Vanilla JavaScript** — particle canvas, dark mode, scroll animations, scrolling text

## ✨ Features

- **Animated particle canvas background** — full-page floating particle animation rendered on `<canvas>`
- **Dark mode toggle** — moon icon button switches the entire page between light and dark themes
- **Scrolling offers bar** — yellow CSS marquee announcing seasonal offers and discounts
- **Sticky navbar** — with smooth scroll links to Fruits, Vegetables, Bales and Contact sections
- **Hero section** — zoom-in animated card with CTA "Explore Products" button
- **Why Choose Us** — 3 feature cards (100% Organic, Fast Delivery, Premium Quality) with hover-lift effect
- **Categories section** — 3 category cards (Fruits, Vegetables, Bales) with icon and "Explore" CTA
- **Featured Products grid** — 4 product cards with price, "Sale" badge, and "View Details" button
- **"Farm to Table" process** — 4-step icon timeline (Harvest → Quality Check → Packaging → Delivery)
- **Bales section** — feature list (Sustainable Materials, High Durability, Various Sizes) with "View Catalog" CTA
- **Testimonials** — 3 five-star review cards from customers and farm managers
- **Contact form** — Name, Email, Message fields with submit button
- **Scroll-reveal animations** — fade-up and zoom-in effects triggered on scroll
- Dark footer with social icons (Facebook, Twitter, Instagram)
- Fully responsive across all screen sizes
